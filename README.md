## SharePlate

SharePlate Connect is a community-focused food donation platform that helps restaurants, caterers, event organizers, and families find NGOs, orphanages, and old-age homes that can accept surplus food.

The idea is simple: good food should reach people who need it instead of going to waste.

## What the project does

Donors can share information about available food, including the quantity, preparation time, pickup area, and preferred recipient type. SharePlate then suggests a suitable recipient and provides contact and direction details so both sides can coordinate directly.

Recipient organizations can also register themselves through the website. Their information is saved immediately and displayed in the public recipient directory.

## Features

- Browse NGOs, orphanages, and old-age homes
- Filter recipients by organization type
- Register a new recipient organization
- Save food donation requests and suggest a matching recipient
- Search by area using OpenStreetMap
- Open directions through Google Maps or OpenStreetMap links
- Store and retrieve all application data using MongoDB
- Responsive layout for desktop and mobile devices
- Basic input validation and secure response headers

## Tech stack

- HTML5
- CSS3
- Vanilla JavaScript
- Node.js
- MongoDB
- OpenStreetMap Nominatim for area lookup

No frontend framework is required. The Node.js server serves the website and provides the API used by the forms and recipient cards.
